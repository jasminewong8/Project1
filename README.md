# How to Play

Click inside any tile to start the game

Player 1 is "O" and Player 2 - AI is "X"

Push "Play Again" to start a new game.

# User Story

AS a gamer

I WANT to play a game of Tic Tac Toe

SO that I can enjoy my time with other people.
